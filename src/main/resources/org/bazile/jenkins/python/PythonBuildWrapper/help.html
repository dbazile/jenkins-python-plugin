Adds a Python installation with pip virtualenv/venv to the PATH of the build.

It is important to remember that the bare Python installation is immutable; you need to create a virtual environment if you'll pip install things.

Usage

In your build script, include something like the following:

#
# Python 3
#

python3 -m venv venv
. venv/bin/activate

python --version

pip install -r requirements.txt



#
# Python 2
#

python2 -m virtualenv virtualenv
. virtualenv/bin/activate

python --version

pip install -r requirements.txt
